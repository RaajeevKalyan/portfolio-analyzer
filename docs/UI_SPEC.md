# UI Specification - Portfolio Risk Analyzer

## Design Principles

### Visual Identity
- **Clean & Professional**: White background (light theme), minimal clutter
- **Data-Focused**: Charts and tables are the hero elements
- **Clear Hierarchy**: Important metrics prominent, details accessible
- **Trust & Reliability**: Professional color palette, consistent spacing

### Typography
```
Font Family: 
  - Primary: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif
  - Monospace: 'Fira Code', 'Courier New', monospace (for numbers/data)

Font Sizes:
  - H1: 32px / 2rem (Page titles)
  - H2: 24px / 1.5rem (Section headers)
  - H3: 20px / 1.25rem (Card titles)
  - Body: 16px / 1rem (Default text)
  - Small: 14px / 0.875rem (Captions, metadata)
  - Tiny: 12px / 0.75rem (Labels, footnotes)

Font Weights:
  - Regular: 400
  - Medium: 500
  - Semibold: 600
  - Bold: 700
```
